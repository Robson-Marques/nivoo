# 🚀 INICIAR AGORA - NFC-e DELIVERY PRO

## ⚡ 3 Passos Rápidos

### 1. Iniciar Servidor
```bash
npm run dev
```
✅ Acesse: http://localhost:8089/

### 2. Login Admin
Use suas credenciais administrativas existentes

### 3. Menu Lateral
Procure por:
- 📋 **Fiscal (NFC-e)** ← Configure seus dados
- 📊 **Histórico NFC-e** ← Veja notas emitidas

---

## 🎯 Próximo: Configurar Dados Fiscais

**Clique em: Menu → Fiscal (NFC-e)**

Preencha 4 abas:
1. **Empresa** - CNPJ, IE, etc
2. **Endereço** - Localização
3. **Certificado** - Upload .pfx
4. **Ambiente** - Homologação/Produção

Salve e pronto! ✨
